===================================================
VIX 檢視器 離線版本 (Viewer.exe) 使用說明
===================================================

這個 `VIX_Viewer.exe` 是包含所有前端與後端邏輯的「獨立離線版」，完全不需要安裝 Python 環境或上網即可執行。

【使用步驟】
1. 將本目錄下的 `dist/VIX_Viewer.exe` 複製出來，放到一個新的資料夾中（例如隨身碟上的 `VIX_Demo`）。
2. 將專案根目錄下的 `output` 和 `資料來源` 資料夾（針對您要 Demo 的特定日期）一起複製，和 `VIX_Viewer.exe` 放在**同一層目錄**。
   資料結構如下範例：
   VIX_Demo/
     ├── VIX_Viewer.exe
     ├── output/
     │     └── 20251231/
     └── 資料來源/
           └── 20251231/

3. 雙擊執行 `VIX_Viewer.exe`。
4. 程式會自動開啟一個小黑窗（本機伺服器）並在瀏覽器中開啟 VIX 觀測站網頁。
5. 在右上角的「選擇日期」中挑選您的 Demo 日期（如 20251231）即可完整顯示報表及分析互動功能。

注意：請勿關閉那個黑色的伺服器視窗，直到您檢視完畢。
